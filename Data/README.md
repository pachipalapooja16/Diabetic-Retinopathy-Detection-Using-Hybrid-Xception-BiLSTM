# Dataset

This project uses the Indian Diabetic Retinopathy Image Dataset (IDRiD).

Dataset source:

https://ieee-dataport.org/open-access/indian-diabetic-retinopathy-image-dataset-idrid

The raw dataset is not included in this repository because medical image datasets can be large and may have usage restrictions.

To run the notebook, download the IDRiD dataset from the official source and place the image files and label files in the required local or Google Drive path mentioned in the notebook.

Expected dataset components:

* Retinal fundus images
* Diabetic retinopathy grading labels
* Risk of macular edema labels

After downloading the dataset, update the dataset path in the notebook before running the code.
